## Cognitive Load Reduction

### OLD: High Cognitive Load

```
User must:
1. Read each description
2. Remember which tasks are which
3. Scroll frequently
4. Re-orient after scrolling
5. Process lots of text

Result: Slower, more tiring
```

### NEW: Low Cognitive Load

```
User can:
1. Scan titles quickly
2. Identify tasks by title + icon
3. See everything at once
4. Collapse what's not relevant
5. Click for details only when needed

Result: Faster, less tiring
```
